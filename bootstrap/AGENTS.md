# AGENTS.md — NEW PROJECT BOOTSTRAP

Ты находишься в **новом проекте**, который нужно настроить в Cursor.

В этой рабочей папке также может присутствовать подпапка `cursor_guide/` — это **библиотека/гайд**. Не воспринимай её как основной проект.

## Главная цель
1. Понять, **что это за проект** (короткое интервью).
2. Настроить Cursor **для этого проекта**: `.cursor/`, `.cursorignore`, Rules, Notepads и т.д.
3. Делать изменения **в корне проекта**, а не внутри `cursor_guide/`.

## Важно
- Не редактируй файлы внутри `cursor_guide/`, если пользователь явно не попросил улучшить сам гайд.
- Используй `cursor_guide/cursor-project-setup-guide.md` как источник пошаговых инструкций.

## Интервью (задай вопросы и дождись ответов)
Спроси минимум:
1. Тип проекта: (web/backend/mobile/cli/library/docs/monorepo)?
2. Стек: язык(и), фреймворк, сборщик (если есть).
3. Планируется ли публичный репозиторий? (GitHub/GitLab/нет)
4. Нужны ли линтер/форматер/тесты? (предпочтения)
5. Требования по приватности (Privacy Mode vs Legacy)?
6. CI/CD: будет ли GitHub Actions и какие проверки (lint/test/build)?

## Дальше (после ответов)
- Создай/обнови `.cursor/` и `.cursorignore` в корне проекта.
- Заполни `.cursor/rules/project.mdc` под стек проекта.
- Создай/обнови Notepads в `.cursor/notepads/` под проект.
