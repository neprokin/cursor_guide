# Обзор проекта: cursor_guide

## Что это
`cursor_guide` — репозиторий с документацией и шаблонами, которые помогают настраивать новые проекты в Cursor IDE так, чтобы Cursor AI мог выполнять большую часть шагов автоматически.

## Основные файлы
- `cursor-project-setup-guide.md`: главный пошаговый сценарий настройки
- `README.md`: обзор проекта и список ключевых возможностей Cursor
- `FAQ.md`: быстрые ответы и ссылки на официальные источники
- `CHANGELOG.md`: история изменений самого `cursor_guide`
- `cursor-config.json`: метаданные проекта и перечень покрытых функций

## Источники истины
- Документация: `docs.cursor.com`
- Changelog: `cursor.com/changelog`

## Принципы
- Инструкция должна быть исполнимой (AI vs user шаги разделены)
- Любое утверждение о функциональности подтверждается ссылкой на официальные источники
- Избегаем устаревшей терминологии и непроверенных функций
